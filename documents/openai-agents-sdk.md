# Building with the OpenAI Agents SDK

## High-level Agent Architecture

- Foundation LLM: GPT4.0, Claude 3.5 Sonnet, Liama
- Orchestrator: ChatGPT, Copilot etc.
- Knowedge: e.g. Your documents, The Web, Instructions.
- Autonomy: Planning, exception, self-learning.
- User Experience: e.g. A website, Microsoft Teams, Your app.


## Flow

Input ->

- Agent1 - Developer
- Agent2 - Code reviewer
- Agent3 - Lead Developer

-> Final Output
